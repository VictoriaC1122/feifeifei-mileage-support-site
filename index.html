1. 本 app 提供的內容

「飛飛飛 - 航空哩程計算器」主要提供航空公司常旅客計畫、航空哩程兌換規則、 信用卡轉點資訊與航線比較整理，協助使用者更有效率地理解不同哩程方案。

2. 資訊來源說明

本 app 所呈現的航空哩程、票規與部分轉點資訊，係基於各航空公司與相關合作夥伴公開揭露的資料進行整理。

因航空公司票規、動態定價、信用卡轉點活動與合作內容可能隨時調整，實際兌換條件仍應以各官方網站、官方公告或官方客服說明為準。

3. 我們蒐集哪些資料

截至目前版本，app 主要用於展示與比較哩程資訊，不要求使用者建立帳號，也不以此頁面收集姓名、電話或付款資料。

若 app 未來加入本機偏好設定、搜尋條件保存或通知功能，可能僅在裝置本機保存相關資料。

4. 資料如何使用

  • 顯示與整理航空哩程計畫、規則與航線比較資訊。
  • 協助使用者依需求篩選艙等、區域、航線或信用卡轉點關係。
  • 改善 app 的閱讀體驗、資訊呈現與產品支援回覆品質。

5. 是否上傳到外部伺服器

目前我們沒有在這個支援與隱私權頁面中提供帳號系統，也不在此頁面主動蒐集使用者輸入資料。

若 app 本體未來新增需要網路連線的資料更新或外部 API 服務，相關資料流程將於版本更新時同步修訂本政策。

6. 第三方服務與追蹤

截至本政策版本，我們不以本頁面進行廣告投放、第三方登入或行為追蹤用途。

若未來導入分析工具、外部資料 API 或其他第三方服務，將在本頁更新相應說明。

7. 資料分享與揭露

我們不會出售或出租使用者個人資料。

若依法令要求、配合主管機關合法程序，或為保護使用者、開發者與公眾安全而有必要時，才可能依法提供必要資訊。

8. 使用者權利

  • 你可以隨時停止使用本 app。
  • 若對資料來源、使用方式或隱私權政策有疑問，可直接透過聯繫方式來信詢問。
  • 若未來 app 提供本機儲存內容，你可透過刪除 app 或清除資料來移除相關資訊。

9. 政策更新

若 app 功能、資料處理流程或第三方服務有重大調整，本頁內容將一併更新。
